2주차

HTML, CSS 소개 및 실습 1 (github를 활용한 버전관리)

  • 에디터로는 Subline Text 활용
    드림위버는 위지윅기반의 에디터이어서 보면서 코딩하기는 좋지만 하드코딩(소스코드 그자체)을 할 경우에는 전문에디터가 더 편합니다.
  • GIT 버전콘트롤 서비스 www.github.com
  • GIT 호스팅 pages.github.com
  • 제작 샘플 sample1.html
    샘플 파일 sample1.html 파일을 만들고 제작절차 에 따라 제시된 예제 샘플의 코드를 따라서 작성해 보세요.
    1. 윈도우 탐색기에서 C:\dev 폴더를 생성한다. (프로젝트들 하위로 생성하기 위함. 이름은 각자 알아서...)
    2. Google 에서 sublime text 를 검색하여 에디터를 설치
      • Project > Add folder to project 메뉴를 이용하 C:\dev 폴더를 추가한다.
    3. Google 에서 github를 검색하여 회원가입
      • 소스 작성시 Copy & Paste 금지!
      • 무료인 free로 가입을 한다.
      • Repositary를 {username}.github.io 로 생성한다. (http://pages.github.com/).
      • Initialize this repository with a README 를 체크해 준다.
      • Clone in Desktop 버튼을 클릭한 후 Github Desktop 프로그램을 설치한다.
      • Clone in Desktop 프로그램에 로그인
      • 다시 아까 Github사이트에서 자신의 리포지터리의 Clone in Desktop 버튼을 클릭한다.
      • 생성된 자신의 리포지터리에서 이 문서를 참조하여 sample1.html 을 생성한다.
      • Github 프로그램에서 변경된 파일을 Commit 하고 우측상단에 Sync를 클릭합니다.
      • {username}.github.io/sample1.html 파일을 웹브라우져에서 확인한다.
  • 샘플제작에 필요한 태그 및 CSS 설명
    <html></html>HTML 문서를 만듭니다.
    <head></head>보여지는 것이 아니라 문서에 대한 제목이나 정보들을 정의하는 부분입니다.
    <body></body>HTML 문서에서 보여지는 부분의 문서 내용입니다.
    <hl></hl> <h2></h2> ~ <h6></h6>헤드라인. 숫자가 커질 수록 작아집니다.
    <b></b>볼드체
    <i></i>이탤릭체
    <a href="URL"></a>링크걸기(클릭하면 문서 이동)
    <p></p>문장
    <ol></ol>뷸렛 리스트
    <ul></ul>번호 리스트
    <li></li>리스트 항목
    <img src="name">이미지 삽입
    <hr />수평선
    <table></table>테이블
    <tr></tr>row 줄
    <td></td>column 한 칸(내용)
    <th></th>column 한 칸(제목)
    <frame src="URL">외부 문서를 현재 문서 안에 삽입한다.
